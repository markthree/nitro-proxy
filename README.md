# nitro-proxy

前端静态服务，支持自动 `api` 代理

<br />

## 动机

想要快速进行前端的部署，而不用花费时间进行 `api` 代理

<br />

## 使用

### 打包

在你的 `vite` 项目中

```shell
# 生成自动代理的后端服务
npx nitro-proxy
```

### 启动

打包完成后启动服务

```shell
npx nitro-proxy start
```
